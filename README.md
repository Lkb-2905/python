# IP Geolocation (python)

Ce dépôt contient le projet d'IP Geolocation dans le sous-dossier `python/`.

Voir la documentation complète et les instructions d'exécution dans :

- `python/README.md`

Commandes rapides :

```powershell
cd python
poetry install
poetry run uvicorn webserv:app --reload
poetry run python client.py
```
